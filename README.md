# ComfyUI-diffusiondb
This node gives you the quick access to 14 million text-to-image prompts from the [DiffusionDB](https://huggingface.co/datasets/poloclub/diffusiondb) dataset.

```code
cd ComfyUI/custom_nodes/
git clone https://github.com/rzgarespo/ComfyUI-diffusiondb.git
``` 
Requirements:
```python
pip install duckdb
``` 

Restart ComfyUI and search for SDprompt. You can use it standalone or connect it to your workflow.

![scsh](https://cdn-uploads.huggingface.co/production/uploads/644717ea27a3e9e0b7c7207a/I_KcjskfaJpaNOinA_Lbd.png)


Example:
```text
3d octane render ultra photo realistic image, beautiful planet monumental organic ceramic white wooden occult robot samourai cyberpunk, on the art contemporary gallery art by mucha hirschl
```
